#### Запуск проекта

Для запуска проекта нужно в каталоге проекта выполнить команду
    
    docker-compose up
    
Таблицы и данные будут автоматически созданы с помощью скриптов в каталоге `init`

Параметры проекта:

    Субд - MySql 8
    Название базы - shop
    Логин - root
    Пароль - secret
    Порт - 33100
